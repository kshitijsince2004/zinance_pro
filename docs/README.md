## Project Structure
```
src/
├── components/          # Reusable UI components
├── pages/              # Page components for routing
├── lib/                # Business logic and utilities
├── types/              # TypeScript type definitions
├── hooks/              # Custom React hooks
├── contexts/           # React context providers
└── docs/               # Documentation files
```

## Architecture Overview
The FAMS system follows a modular architecture with:
- **Frontend**: React with TypeScript for type safety
- **State Management**: React Context and local storage
- **Styling**: Tailwind CSS with custom design tokens
- **Data Layer**: Service layer pattern for business logic
- **Backend Ready**: Designed for C# Web API integration

## Key Features
- Asset lifecycle management
- Depreciation calculations (SLM, WDV, etc.)
- QR code generation and scanning
- Import/Export functionality
- Verification and audit trails
- AMC and warranty tracking
- Advanced reporting and analytics
