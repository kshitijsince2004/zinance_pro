# Zinance - Fixed Asset Management Software

Enterprise Fixed Asset Management Platform built with React, TypeScript, and Vite.

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **UI Components**: shadcn/ui + Radix UI
- **Styling**: Tailwind CSS
- **State Management**: TanStack Query
- **Routing**: React Router DOM
- **Forms**: React Hook Form + Zod
- **Charts**: Recharts
- **Backend**: Supabase

## Getting Started

### Prerequisites

- Node.js 18+ or Bun

### Install dependencies

```sh
npm install
# or
bun install
```

### Run locally

```sh
npm run dev
# or
bun dev
```

The app will be available at `http://localhost:8080`.

## Building for Production

```sh
npm run build
```

## Linting

```sh
npm run lint
```
